# PaintChart

PaintChart는 트레이딩 뷰 차트의 테마를 모아놓은 사이트이다.
비슷비슷한 흰바탕에 초록양봉과 빨간음봉 차트가 아닌, 오써들의 개성있고 가독성있는 차트테마로 사람들에게 좋아요와 명성을 많이 얻는게 도와주는게 목표.
